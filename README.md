## What is Freelancer OS?

Freelancer OS is a complete **Notion business template** built for freelancers, consultants, and solo founders. It replaces the chaos of spreadsheets, sticky notes, and disconnected tools with one organised workspace.

5 fully connected databases — every piece of your business in one place, all linked together.

---

## What's inside

| Section | What it does |
|---|---|
| 👥 **Clients CRM** | Track leads, active clients, completed — with revenue and contact info |
| 📁 **Projects** | All projects linked to clients, with budget, deadline, priority, status |
| ✅ **Tasks** | Daily tasks linked to projects — kanban + list view |
| 🧾 **Invoices** | Draft → Sent → Paid → Overdue tracking |
| ⏱️ **Time Log** | Billable hours per project with hourly rate calculations |
| 📊 **Dashboard** | Weekly snapshot: active work, hours, invoices, revenue |
